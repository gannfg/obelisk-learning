# **Superteam Study**
by Obelisk Protocol

**Superteam Study** is a structured learning and collaboration platform designed to help developers and teams learn Web3, build real projects, participate in workshops, and qualify for grants through measurable engagement.

The platform combines **semester-based classes**, **missions**, **projects**, **workshops**, **gamification**, and **AI mentorship** into one unified learning experience.
